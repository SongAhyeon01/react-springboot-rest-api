## 프로젝트 소개 😎
React로 만들어진 Front End가 정해져있는 상황에서,
백엔드 개발자가 Spring Boot로 상품관리 API를 구현하여 A-Z 최종 서비스를 완성시켜봅니다.
클로닝 외에도 "본인만의 아이디어"를 추가하여 더 발전시켜 완성해봅니다.
